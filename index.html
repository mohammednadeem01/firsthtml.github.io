TAG PURPOSE
< p > < /p> This tag defines a paragraph
< html> < /html> This is the root of HTML document
< h>< /h > This is used to define the heading of the document
< em>< /em > It is used to render text as emphasized
< b>< /b> It is used to specify bold content
< i>< /i> It is used to write the content in italic format
< u>< /u> It is used to set the content underline
< strike > It is used to represent as deleted text
< mark>< /mark> It is used to highlight text
< li>< /li> It is used to list the content
< !-- Statements... --> It is used to set the comment in html document. It is not visible on the browser
< font > It is used to specify the font size, font color and font-family in html document
< br> It is used to break the line
